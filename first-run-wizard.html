Willkommen bei Erfinder!

Kinderfreundlich und sicher

Erfinder ist ein spezieller Browser für Kinder, der automatisch gefährliche oder ungeeignete Inhalte blockiert.

🔍 Automatische Inhaltsprüfung mit FragFinn
🔒 Eltern-PIN für Sicherheitseinstellungen
⭐ Whitelist für vertrauenswürdige Websites
🔐

Eltern-PIN einrichten

Bitte wählen Sie eine 4-stellige PIN für den Elternbereich. Diese PIN wird benötigt, um Sicherheitseinstellungen zu ändern und Websites zur Whitelist hinzuzufügen.

⭐

Vertrauenswürdige Websites hinzufügen

Sie können optional bereits jetzt einige vertrauenswürdige Websites zur Whitelist hinzufügen. Diese werden automatisch zugelassen, auch wenn sie nicht bei FragFinn gelistet sind.

    Hinweis: Sie können jederzeit später weitere Websites zur Whitelist hinzufügen, indem Sie den Elternbereich öffnen.

    ✅

    Einrichtung abgeschlossen!

    Erfinder ist jetzt einsatzbereit. Ihr Kind kann sicher im Internet surfen.

    Was wurde eingerichtet:

    • ✅ Eltern-PIN wurde festgelegt
    • ✅ Whitelist wurde konfiguriert
    • ✅ Automatische Inhaltsprüfung ist aktiviert

    Zugriff auf Elternbereich: Klicken Sie auf das 🔒 Symbol in der oberen rechten Ecke des Browsers.

    Whitelist erweitern: Wenn eine blockierte Website sicher ist, können Sie sie über den Elternbereich zur Whitelist hinzufügen.
